## 🚨 **CRITICAL ISSUES TO AVOID (Based on Real Failures)**

### **1. Python 3.13 Compatibility Issues**
**❌ WRONG:** `cryptography==41.0.8` (version not available)
**✅ CORRECT:** `cryptography>=3.4.0,<45.0.0`
**✅ CORRECT:** Use version ranges for all dependencies

### **2. FFmpeg Dependency Management**
**❌ CRITICAL ERROR:** Not mentioning FFmpeg requirement in setup
**✅ SOLUTION:** MUST include FFmpeg installation in README:
```powershell
# PowerShell (as Admin):
choco install ffmpeg -y
# OR manual: Download from ffmpeg.org, add to PATH
```

### **3. Missing Flask Dependencies**
**❌ WRONG:** Not including all required Flask components
**✅ CORRECT:** Must include all in requirements.txt:
- Flask
- Jinja2
- MarkupSafe
- itsdangerous
- Click
- blinker

### **4. Regex Character Range Errors**
**❌ WRONG:** `filename = re.sub(r'[^\w\s-.]', '', filename)`
**✅ CORRECT:** `filename = re.sub(r'[^\w\s\-\.]', '', filename)`
**⚠️ ALWAYS escape special characters in regex patterns**

### **5. Virtual Environment Setup**
**❌ WRONG:** Not providing PowerShell execution policy fix
**✅ CORRECT:** Include these commands:
```powershell
Set-ExecutionPolicy -ExecutionPolicy RemoteSigned -Scope CurrentUser
.\venv\Scripts\Activate.ps1
```

### **Input Validation:**
- **URL Validation:** Check YouTube URL format, extract video ID
- **Security Checks:** Block localhost, IP addresses, non-standard ports
- **Sanitization:** Remove injection characters, null bytes
- **Length Limits:** Max URL length, filename length restrictions

### **Rate Limiting:**
- **Implementation:** Flask-Limiter with memory storage
- **Limits:** 100 conversions per day, 20 per hour, 5 per minute
- **IP-based:** Use client IP for rate limiting

### **File Security:**
- **Size Limits:** Maximum 50MB file size
- **Type Validation:** Only allow MP3, WAV, M4A extensions
- **Temporary Cleanup:** Auto-cleanup after 1 hour
- **Secure Filenames:** Sanitize all filenames

### **Web Security:**

- **Security Headers:** X-Content-Type-Options, X-Frame-Options, etc.
